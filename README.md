### 1. Clone the repository
```bash
git clone https://github.com/Popal999/Popal999.github.io.git
```

### 2. Navigate to the project
```bash
cd Popal999.github.io
```

### 3. Open the website
Open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx http-server
```

Visit: `http://localhost:8000`

## 📝 Usage | الاستخدام

### Customization | التخصيص

<div dir="rtl">

#### تعديل المحتوى
1. افتح `index.html` لتعديل المحتوى الرئيسي
2. عدّل الأنماط في `css/style.css`
3. أضف وظائف JavaScript في `js/main.js`

#### تغيير الألوان
يمكنك تعديل الألوان الأساسية في `css/style.css`:

</div>

```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
  --accent-color: #your-color;
}
```
